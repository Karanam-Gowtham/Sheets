01

Sliding Window

Array · String ▾

Maintain a window (subarray/substring) that slides over data. Expand right, shrink left based on a condition. Avoids nested loops → O(n) instead of O(n²).

Use When

  • Contiguous subarray/substring problem
  • Max/min sum of fixed-length window
  • Longest substring with constraint
  • Keywords: "subarray", "window", "consecutive"
⏱ Time: O(n) 💾 Space: O(1) or O(k)
Java — Fixed Window (Max Sum)
int maxSum(int[] arr, int k) {
  int sum = 0, max = 0;
  // first window
  for (int i = 0; i < k; i++)
    sum += arr[i];
  max = sum;
  // slide
  for (int i = k; i < arr.length; i++) {
    sum += arr[i] - arr[i - k];
    max = Math.max(max, sum);
  }
  return max;
}
Java — Variable Window (Longest Substring)
int longestK(String s, int k) {
  Map<Character,Integer> freq = new HashMap<>();
  int l = 0, max = 0;
  for (int r = 0; r < s.length(); r++) {
    freq.merge(s.charAt(r), 1, Integer::sum);
    while (freq.size() > k) {  // shrink
      char c = s.charAt(l++);
      freq.merge(c, -1, Integer::sum);
      if (freq.get(c) == 0) freq.remove(c);
    }
    max = Math.max(max, r - l + 1);
  }
  return max;
}

02

Two Pointers

Array · String ▾

Two indices move toward each other (or same direction) to find pairs/triplets satisfying a condition. Works on sorted arrays or when iterating two sequences simultaneously.

Use When

  • Sorted array, find pair with target sum
  • Remove duplicates / partition array in-place
  • Palindrome check, reverse string
  • Container with most water type problems
⏱ Time: O(n) 💾 Space: O(1)
Java — Two Sum (Sorted Array)
int[] twoSum(int[] a, int target) {
  int l = 0, r = a.length - 1;
  while (l < r) {
    int s = a[l] + a[r];
    if (s == target)
      return new int[]{l, r};
    else if (s < target) l++;
    else r--;
  }
  return new int[]{-1, -1};
}

Prefix Sum

Array ▾

Precompute cumulative sums so any range sum query [l,r] = prefix[r] - prefix[l-1] is answered in O(1). Build the prefix array once, answer many queries fast.

Use When

  • Multiple range sum queries on same array
  • Subarray sum equals K (with hashmap)
  • Count subarrays with specific property
⏱ Build: O(n) · Query: O(1) 💾 Space: O(n)
Java — Subarray Sum Equals K
int subarraySum(int[] nums, int k) {
  Map<Integer,Integer> map = new HashMap<>();
  map.put(0, 1); // empty prefix
  int sum = 0, count = 0;
  for (int n : nums) {
    sum += n;
    count += map.getOrDefault(sum - k, 0);
    map.merge(sum, 1, Integer::sum);
  }
  return count;
}

04

Monotonic Stack

Stack ▾

Stack that maintains elements in monotonically increasing or decreasing order. Pop when incoming element breaks the order — the popped element's answer is the current element.

Use When

  • Next Greater / Smaller Element
  • Largest Rectangle in Histogram
  • Daily Temperatures, Stock Span
⏱ Time: O(n) 💾 Space: O(n)
Java — Next Greater Element
int[] nextGreater(int[] arr) {
  int[] res = new int[arr.length];
  Arrays.fill(res, -1);
  Deque<Integer> stack = new ArrayDeque<>();
  for (int i = 0; i < arr.length; i++) {
    while (!stack.isEmpty() &&
           arr[stack.peek()] < arr[i]) {
      res[stack.pop()] = arr[i]; // found NGE
    }
    stack.push(i);
  }
  return res;
}

05

Binary Search

Array ▾

Eliminate half the search space each step. Beyond finding a value, binary search applies to "search on answer" — binary search over the answer space when a monotone condition exists.

Use When

  • Sorted array, find target
  • "Minimum possible maximum" / "Maximum possible minimum"
  • Condition is monotone: false…false→true…true
⏱ Time: O(log n) 💾 Space: O(1)
Java — Universal Template
int binarySearch(int[] arr, int target) {
  int l = 0, r = arr.length - 1;
  while (l <= r) {
    int mid = l + (r - l) / 2; // avoid overflow
    if (arr[mid] == target) return mid;
    else if (arr[mid] < target) l = mid + 1;
    else r = mid - 1;
  }
  return -1; // l = insertion point
}

// Search on Answer template
int lo = minAns, hi = maxAns;
while (lo < hi) {
  int mid = (lo + hi) / 2;
  if (feasible(mid)) hi = mid;
  else lo = mid + 1;
}

06

Fast & Slow Pointers

LinkedList ▾

Two pointers move at different speeds (1x and 2x). Slow reaches middle when fast reaches end. If they ever meet, there's a cycle. Also used to find cycle start.

Use When

  • Detect cycle in LinkedList / array
  • Find middle of LinkedList
  • Find cycle start (Floyd's algorithm)
  • Happy number problem
⏱ Time: O(n) 💾 Space: O(1)
Java — Detect Cycle + Find Start
boolean hasCycle(ListNode head) {
  ListNode slow = head, fast = head;
  while (fast != null && fast.next != null) {
    slow = slow.next;
    fast = fast.next.next;
    if (slow == fast) return true;
  }
  return false;
}

// Find cycle start (Floyd's algorithm)
ListNode cycleStart(ListNode head) {
  ListNode slow = head, fast = head;
  while (fast.next != null && fast.next.next != null) {
    slow = slow.next; fast = fast.next.next;
    if (slow == fast) {
      slow = head; // reset slow to head
      while (slow != fast) { slow = slow.next; fast = fast.next; }
      return slow; // cycle start
    }
  }
  return null;
}

In-place Reversal of LinkedList

LinkedList ▾

Reverse a linked list (or a sub-portion) without extra space by relinking pointers. Use prev/curr/next variables.

⏱ Time: O(n)💾 Space: O(1)
Java — Reverse Between (positions)
ListNode reverseBetween(ListNode head, int l, int r) {
  ListNode dummy = new ListNode(0, head);
  ListNode pre = dummy;
  for (int i = 0; i < l - 1; i++) pre = pre.next;
  ListNode curr = pre.next;
  for (int i = 0; i < r - l; i++) {
    ListNode next = curr.next;
    curr.next = next.next;
    next.next = pre.next;
    pre.next = next;
  }
  return dummy.next;
}

Tree & Graph Patterns
08

Tree BFS (Level Order)

Tree · Graph ▾

Process nodes level by level using a queue. Start with root, enqueue children. Great for shortest path in unweighted graphs and anything level-related.

Use When

  • Level-order traversal
  • Minimum depth / shortest path
  • Right view, zigzag traversal
  • Rotten Oranges, 0-1 Matrix
⏱ Time: O(n)💾 Space: O(w) — max width
Java — Level Order Traversal
List<List<Integer>> levelOrder(TreeNode root) {
  List<List<Integer>> res = new ArrayList<>();
  if (root == null) return res;
  Queue<TreeNode> q = new LinkedList<>();
  q.offer(root);
  while (!q.isEmpty()) {
    int size = q.size(); // current level size
    List<Integer> level = new ArrayList<>();
    for (int i = 0; i < size; i++) {
      TreeNode node = q.poll();
      level.add(node.val);
      if (node.left != null) q.offer(node.left);
      if (node.right != null) q.offer(node.right);
    }
    res.add(level);
  }
  return res;
}

Tree DFS

Tree · Graph ▾

Explore as deep as possible before backtracking. Three flavors: Pre-order (root→L→R), In-order (L→root→R), Post-order (L→R→root). Recursive or iterative with explicit stack.

Use When

  • Path sum problems
  • Serialize/Deserialize tree
  • Lowest Common Ancestor
  • Validate BST
⏱ Time: O(n)💾 Space: O(h) — height
Java — Path Sum II
void dfs(TreeNode node, int target, int curr,
         List<Integer> path, List<List<Integer>> res) {
  if (node == null) return;
  path.add(node.val);
  curr += node.val;
  if (node.left == null && node.right == null
      && curr == target)
    res.add(new ArrayList<>(path));
  dfs(node.left, target, curr, path, res);
  dfs(node.right, target, curr, path, res);
  path.remove(path.size() - 1); // backtrack
}

10

Union Find (Disjoint Set)

Graph ▾

Track which elements belong to the same connected component. Union merges two sets. Find returns the root/representative. Path compression + union by rank makes it nearly O(1) amortized.

⏱ Time: O(α(n)) ≈ O(1)💾 Space: O(n)
Java — Union Find Template
class UnionFind {
  int[] parent, rank;
  UnionFind(int n) {
    parent = new int[n]; rank = new int[n];
    for (int i = 0; i < n; i++) parent[i] = i;
  }
  int find(int x) { // path compression
    if (parent[x] != x) parent[x] = find(parent[x]);
    return parent[x];
  }
  boolean union(int x, int y) {
    int px = find(x), py = find(y);
    if (px == py) return false; // already connected
    if (rank[px] < rank[py]) parent[px] = py;
    else if (rank[px] > rank[py]) parent[py] = px;
    else { parent[py] = px; rank[px]++; }
    return true;
  }
}

Merge Intervals

Intervals ▾

Sort intervals by start time. Merge overlapping intervals by checking if current start ≤ previous end. Variants: insert interval, non-overlapping intervals.

Use When

  • Overlapping intervals (meeting rooms)
  • Insert/remove intervals
  • Minimum platforms/workers needed
⏱ Time: O(n log n)💾 Space: O(n)
Java — Merge Intervals
int[][] merge(int[][] intervals) {
  Arrays.sort(intervals, (a,b) -> a[0] - b[0]);
  List<int[]> res = new ArrayList<>();
  int[] cur = intervals[0];
  for (int[] iv : intervals) {
    if (iv[0] <= cur[1])          // overlap
      cur[1] = Math.max(cur[1], iv[1]);
    else { res.add(cur); cur = iv; }
  }
  res.add(cur);
  return res.toArray(new int[0][]);
}

Cyclic Sort

Array ▾

When array contains numbers in range [1, n], place each number at its correct index (num-1). In one pass O(n), find missing/duplicate numbers without extra space.

⏱ Time: O(n)💾 Space: O(1)
Java — Find All Missing Numbers
void cyclicSort(int[] arr) {
  int i = 0;
  while (i < arr.length) {
    int j = arr[i] - 1; // correct index
    if (arr[i] != arr[j])
      swap(arr, i, j);  // put in right place
    else i++;
  }
}
// After sort: positions where arr[i]!=i+1 are missing

13

Top K Elements

Heap ▾

Use a min-heap of size K to track the K largest elements. When heap size exceeds K, remove the minimum. Result: K largest remain. Flip for K smallest with max-heap.

Use When

  • K largest/smallest elements
  • K most frequent elements
  • Kth largest in stream
  • Closest K points to origin
⏱ Time: O(n log k)💾 Space: O(k)
Java — K Largest Elements
int[] kLargest(int[] nums, int k) {
  PriorityQueue<Integer> minH = new PriorityQueue<>();
  for (int n : nums) {
    minH.offer(n);
    if (minH.size() > k) minH.poll(); // remove min
  }
  return minH.stream().mapToInt(x->x).toArray();
}

// K most frequent (with bucket sort)
int[] topKFrequent(int[] nums, int k) {
  Map<Integer,Integer> freq = new HashMap<>();
  for (int n : nums) freq.merge(n, 1, Integer::sum);
  PriorityQueue<Map.Entry<Integer,Integer>> pq =
    new PriorityQueue<>((a,b)->a.getValue()-b.getValue());
  for (var e : freq.entrySet()) {
    pq.offer(e);
    if (pq.size() > k) pq.poll();
  }
  return pq.stream().mapToInt(Map.Entry::getKey).toArray();
}

14

Two Heaps

Heap ▾

Maintain a max-heap for the lower half and a min-heap for the upper half. Keep sizes balanced. Median = top of one heap (odd count) or average of both tops (even count).

⏱ Add: O(log n) · Median: O(1)💾 Space: O(n)
Java — Find Median from Data Stream
class MedianFinder {
  PriorityQueue<Integer> lo = new PriorityQueue<>(Collections.reverseOrder());
  PriorityQueue<Integer> hi = new PriorityQueue<>();

  void addNum(int n) {
    lo.offer(n);
    hi.offer(lo.poll()); // balance
    if (lo.size() < hi.size()) lo.offer(hi.poll());
  }
  double findMedian() {
    return lo.size() > hi.size()
      ? lo.peek()
      : (lo.peek() + hi.peek()) / 2.0;
  }
}

15

K-way Merge

Heap ▾

Merge K sorted lists using a min-heap. Push the first element of each list, then pop the minimum and push its successor. Result is globally sorted.

⏱ Time: O(n log k)💾 Space: O(k)
Java — Merge K Sorted Lists
ListNode mergeKLists(ListNode[] lists) {
  PriorityQueue<ListNode> pq =
    new PriorityQueue<>((a,b)->a.val-b.val);
  for (ListNode l : lists)
    if (l != null) pq.offer(l);
  ListNode dummy = new ListNode(0), cur = dummy;
  while (!pq.isEmpty()) {
    cur.next = pq.poll();
    cur = cur.next;
    if (cur.next != null) pq.offer(cur.next);
  }
  return dummy.next;
}

Recursion, DP & Backtracking
16

Subsets / Backtracking

Recursion ▾

Generate all subsets / permutations / combinations by making a choice (include / exclude), recurse, then undo (backtrack). Decision tree has 2^n or n! leaves.

Use When

  • All subsets, combinations, permutations
  • N-Queens, Sudoku, Word Search
  • Phone number letter combinations
⏱ Subsets O(2^n) · Perms O(n!)💾 Space: O(n)
Java — Subsets (Backtracking)
List<List<Integer>> subsets(int[] nums) {
  List<List<Integer>> res = new ArrayList<>();
  backtrack(nums, 0, new ArrayList<>(), res);
  return res;
}
void backtrack(int[] nums, int start,
    List<Integer> curr, List<List<Integer>> res) {
  res.add(new ArrayList<>(curr));
  for (int i = start; i < nums.length; i++) {
    curr.add(nums[i]);        // choose
    backtrack(nums, i+1, curr, res); // explore
    curr.remove(curr.size()-1); // unchoose
  }
}

Dynamic Programming

DP ▾

Break problem into overlapping subproblems. Store results (memoization / tabulation). Key insight: optimal solution built from optimal subsolutions.

Use When

  • Overlapping subproblems + optimal substructure
  • Count ways, max/min value, yes/no feasibility
  • Keywords: "minimum cost", "maximum profit", "how many ways"
⏱ Varies: O(n²) · O(n·W) etc.💾 Space: O(n) or O(n²)
Java — 0/1 Knapsack
int knapsack(int[] w, int[] v, int W) {
  int n = w.length;
  int[] dp = new int[W+1];
  for (int i = 0; i < n; i++)
    for (int j = W; j >= w[i]; j--)
      dp[j] = Math.max(dp[j],
               dp[j-w[i]] + v[i]);
  return dp[W];
}
Java — LCS
int lcs(String a, String b) {
  int m=a.length(), n=b.length();
  int[][] dp = new int[m+1][n+1];
  for(int i=1;i<=m;i++)
    for(int j=1;j<=n;j++)
      dp[i][j] = a.charAt(i-1)==b.charAt(j-1)
        ? dp[i-1][j-1]+1
        : Math.max(dp[i-1][j],dp[i][j-1]);
  return dp[m][n];
}

Bitwise XOR

Bit Manipulation ▾

XOR properties: a⊕a=0, a⊕0=a, commutative & associative. XOR all elements → duplicates cancel → missing/single number remains.

Key Properties

  • n & (n-1) → clears lowest set bit
  • n & (-n) → isolates lowest set bit
  • a ⊕ a = 0 (self-cancellation)
  • x << k = x × 2^k | x >> k = x / 2^k
⏱ Time: O(n)💾 Space: O(1)
Java — Bit Tricks
// Single number (XOR all)
int singleNumber(int[] nums) {
  int x = 0;
  for (int n : nums) x ^= n;
  return x;
}
// Count set bits (Brian Kernighan)
int countBits(int n) {
  int count = 0;
  while (n != 0) { n &= n-1; count++; }
  return count;
}
// Check power of 2
boolean isPow2(int n) { return n>0 && (n&(n-1))==0; }
